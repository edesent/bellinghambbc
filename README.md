# Bellingham Bible Baptist Church Website

Next.js 16 / App Router redesign for Bellingham Bible Baptist Church in
Bellingham, Massachusetts.

The site preserves the current logo, homepage background video, additional
parking image, kids photos, livestream/sermons pages, giving page, and the
current public page structure from `https://bellinghambbc.com/`.
